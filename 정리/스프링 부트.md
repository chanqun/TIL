```java
GenericApplicationContext ac = new GenericApplicationContext(); // spring (ioc/di) container
ac.registerBean(HelloController.class); // bean registration
ac.refresh(); // 로딩

이렇게 되면 직접 생성하던 것을 가져와서 사용하면 됨

ac.getBean(HelloController.class);
```
